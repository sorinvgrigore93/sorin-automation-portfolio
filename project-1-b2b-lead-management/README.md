# Sistema de Gestión de Leads B2B y Asignación Comercial

## Descripción
Workflow en n8n Cloud para recibir leads mediante webhook, validar campos obligatorios, normalizar datos y enrutar registros según reglas de negocio.

## Resultados
- Validación automática de campos obligatorios
- Normalización de datos de entrada
- Rutas separadas para casos válidos y no válidos
- Respuestas JSON estructuradas para integraciones consistentes

## Herramientas utilizadas
- n8n
- Webhooks
- JSON
- HTTP Request

## Estado
Proyecto de portfolio orientado a demostrar automatización de procesos e integración de datos.
