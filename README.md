# Sorin Grigore – Portfolio de Automatización

Especialista Junior en Automatización enfocado en workflows con n8n, integraciones con APIs, webhooks, Google Sheets y automatización con IA.

## Proyectos

### 1. [Sistema de Gestión de Leads B2B y Asignación Comercial](./project-1-b2b-lead-management)
Workflow en n8n para recibir leads mediante webhook, validar campos obligatorios, normalizar datos y enrutar registros según reglas de negocio.

### 2. [Sistema de Recepción, Clasificación y Asignación de Tickets con IA](./project-2-ai-ticket-intake)
Workflow en n8n para recibir tickets, estandarizar datos, clasificar solicitudes y derivarlas según su tipo o prioridad.

### 3. [Sistema de Reporte Semanal de Rendimiento Comercial](./project-3-sales-weekly-report)
Automatización programada en n8n para leer datos operativos, calcular métricas semanales y generar resúmenes estructurados para reporting.

## Herramientas y Tecnologías
- n8n
- Webhooks
- APIs REST
- JSON
- HTTP Request
- Google Sheets
- Gmail
- Postman
- OpenAI

## Objetivo
Este repositorio reúne proyectos prácticos de automatización desarrollados para portfolio profesional y búsqueda de oportunidades junior en automatización, operaciones e integración de sistemas.
